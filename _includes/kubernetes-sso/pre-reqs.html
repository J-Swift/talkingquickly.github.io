Pre-requisites

This assumes you have CLI access to a Kubernetes cluster and have both Helm 3 and Kubectl installed and working locally. Finally it assumes that you're using NGINX for Ingress along with cert manager for SSL certificates with a Cluster Issuer called `letsencrypt-production`.

If your configuration is different, the majority of the steps will be the same, but you'll need to change the ingress definitions accordingly.
